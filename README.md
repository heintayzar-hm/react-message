## 💻 Getting Started <a name="getting-started"></a>


To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

<!--
Example command:

```sh
 gem install rails
```
 -->
To install rails and react projects
Rails:

```sh
 git clone https://github.com/heintayzar-hm/messages-generator-rails.git
```

React:
```sh
git clone https://github.com/heintayzar-hm/react-message.git
```

For the Rails,
- Go to your project directory
- run bundler in rails, please go to rails folder, and run(according to your os you might need to change the database in `database.yml`)

```sh
bundle install
rails db:prepare
rails db:setup
```

- Run rails server,

```sh
rails s
```

For the React,
- Go to your Project directory,
- install dependencies and run the server

```s
npm install
npm run dev
```
